Machine Control Side

Procedure 1: Column PC Setup

Configures the primary host machine, Siemens Operate, and SProxy control networks.

Prerequisite: Download Required Files

The Technician needs to go to the Applications Teams Channel and download the whole Tarus Tech folder. This folder has the latest TarusNC version, the latest Default Configuration File for an LCMT 5 axis Baserail, the Software Password Folder for daily passwords, the desktop images used for the two PCs, among many other very useful files and applications.

Critical Warning

Using a default machine configuration on a live machine can cause catastrophic damage. Ensure you have the specific job ConfigurationFiles (e.g., 3529-Column) downloaded before finalizing the software setup.

1

Initial Windows Configuration

  1. Power on the PC. Select United States for the region and US for the keyboard layout. Click Skip for the second layout.
  2. When prompted to connect to a network, select I don't have Internet, then click Continue with limited setup. Accept the Windows License Agreement.
  3. Create a local account using Username: Tarus and Password: Tarus123. (Select the security questions in order and use Tarus as the answer for all 3).
  4. On the Privacy Settings screen, toggle ALL options to OFF and click Accept.
  5. Once at the Windows desktop, connect to the tarusguest Wi-Fi network to gain internet access.
  6. Go to Start > Settings > System > About. Click Rename this PC and name it [JobNum]-Column (e.g., 3529-Column). Select Restart later.
  7. Open the Start menu, type UAC, and press Enter. Drag the slider all the way down to Never notify and click OK.
  8. Go to Settings > Time & Language > Date & time. Change the Time Zone to Eastern Time. Toggle Set time automatically to On, then click Sync now.
  9. Go to Settings > Update & Security. Click Check for updates. Install all available updates (including optional driver updates). Restart and repeat until the page says "You're up to date".
2

WinUtil & Power Plan

  1. Right-click the Start button and select Windows PowerShell (Admin).
  2. Type irm "https://christitus.com/win" | iex and press Enter.
  3. Navigate to the Tweaks tab and click the Standard profile button. Ensure the following additional advanced tweaks are explicitly checked:
    • Create Restore Point
    • Edge Debloat
    • Remove Xbox & Gaming Components
    • Set Classic Right-Click Menu
    • Set Display for Performance
  4. Under the Performance Plans section on the right side of the utility, click Add and Activate Ultimate Performance Profile.
  5. Leave the WinUtil window open. Click back into the blue Admin PowerShell window running behind it, paste the following command, and hit Enter to rebrand the power plan and remove unused ones: 
    $plan = powercfg /list | Select-String "Ultimate"; if ($plan) { $guid = ($plan.ToString() -split '\s+')[3]; powercfg /setactive $guid; powercfg /changename $guid "TARUS - Ultimate Power Plan" "Ultimate Performance for Tarus Control"; powercfg /list | Select-String "Power Scheme GUID" | Where-Object { $_ -notmatch $guid -and $_ -notmatch "8c5e7fda-e8bf-4a96-9a85-a6e23a8c635c" } | ForEach-Object { powercfg /delete ($_ -split '\s+')[3] 2>$null } }
  6. Click Run Tweaks at the bottom of the utility and wait for it to complete.
  7. Navigate to the Updates tab and select Security Settings. (This ensures Windows feature updates are delayed and security updates only occur on Saturdays). Close the utility.
  8. Open the Windows Control Panel > Power Options. Click Change plan settings next to the new TARUS - Ultimate Power Plan. Click Change advanced power settings. Expand Hard disk and ensure "Turn off hard disk after" is set to 0. Click Apply and OK.
3

Network & Software

  1. Open View network connections from the Start menu. Identify the port for the private machine network and rename it to Machine Control. Identify the port for the remote desktop and rename it to Customer Network. Disconnect the Wi-Fi.
  2. Restart the PC and press Esc to enter the BIOS. In the Advanced tab, set "After Power Loss" to Power On. Save and Exit.
  3. Right-click the Machine Control adapter and select Properties. Uncheck everything EXCEPT Internet Protocol Version 4 (TCP/IPv4). Double-click IPv4 and set the IP address to 192.168.214.55 with Subnet Mask 255.255.255.0. Leave Gateway blank. Click Configure > Advanced and set Energy-Efficient Ethernet to Off.
  4. Right-click the Customer Network adapter and select Properties. Ensure IPv4 is set to Obtain an IP address automatically (DHCP). Click Configure > Power Management and uncheck Allow the computer to turn off this device.
  5. Create a C:\Temp folder. Unzip the Siemens installer here. Run SINUMERIK_OPERATE_pc.exe as Administrator. (Select Standard ONE NCU and Complete setup). Restart when prompted.
  6. Open Communication Settings in the Control Panel. Select Access points. Expand S7ONLINE and select the Machine Control adapter.
  7. Run the TarusNC installer as Administrator. (Select TarusNC Only and SProxy). Restart the PC.
4

Final Polish & Access

  1. Navigate to C:\Tarus Products\. Delete the default ConfigurationFiles folder and paste in the specific folder for this job from your downloaded files.
  2. Open the NVIDIA Control Panel > Manage 3D Settings. Set the Global Power Management Mode to Prefer Maximum Performance. Under Program Settings, add TarusNC.exe and set it to High Performance.
  3. Go to Settings > System > Remote Desktop and toggle it to On. Click Confirm.
  4. From your downloaded Tarus Tech folder, copy the TARUS Column PC logo.png file into the Pictures folder. Right-click the Desktop > Personalize, and set it as the Desktop Background.
Operator Station Side

Procedure 2: Desk PC Setup

Configures the client workstation to boot, bypass passwords, and automatically launch the video kiosk.

1

Initial Windows Configuration

  1. Connect the Desk PC to the network via Ethernet. Power on and select United States and US Keyboard.
  2. When prompted to sign in, select Offline account, then click Limited experience.
  3. Create a local account using Username: Tarus and Password: Tarus123. (Select the security questions in order and use Tarus as the answer for all 3).
  4. On the Privacy Settings screen, toggle ALL options to OFF and click Accept.
  5. Go to Start > Settings > System > About. Click Rename PC and enter [JobNum]-Desk.
  6. Go to Settings > Update & Security. Click Check for updates and install all available updates. Restart as needed.
2

WinUtil & System Hardening

  1. Open File Explorer. Right-click the C: drive and select Properties > Security tab > Edit. Check the Allow box for Full Control for ALL listed Users and Groups. Click Apply.
  2. Restart the PC and press Esc to enter the BIOS. In the Advanced tab, set "After Power Loss" to Power On. Save and Exit.
  3. Right-click the Start button and select Windows PowerShell (Admin). Type irm "https://christitus.com/win" | iex and press Enter.
  4. Navigate to the Tweaks tab and click the Standard profile button. Ensure these advanced tweaks are checked:
    • Create Restore Point
    • Edge Debloat
    • Remove Xbox & Gaming Components
    • Set Classic Right-Click Menu
    • Set Display for Performance
  5. Under the Performance Plans section on the right side, click Add and Activate Ultimate Performance Profile.
  6. Paste the following command into the Admin PowerShell window currently running behind WinUtil, then hit Enter to rebrand the power plan and remove unused ones: 
    $plan = powercfg /list | Select-String "Ultimate"; if ($plan) { $guid = ($plan.ToString() -split '\s+')[3]; powercfg /setactive $guid; powercfg /changename $guid "TARUS - Ultimate Power Plan" "Ultimate Performance for Tarus Kiosk"; powercfg /list | Select-String "Power Scheme GUID" | Where-Object { $_ -notmatch $guid -and $_ -notmatch "8c5e7fda-e8bf-4a96-9a85-a6e23a8c635c" } | ForEach-Object { powercfg /delete ($_ -split '\s+')[3] 2>$null } }
  7. Click Run Tweaks at the bottom of the utility and wait for it to complete.
  8. Navigate to the Updates tab and select Disable All Updates. (Critical: This ensures the Kiosk never reboots or lags during production). Close the utility.
  9. Open the Windows Control Panel > Power Options. Click Change plan settings next to the new TARUS - Ultimate Power Plan. Click Change advanced power settings. Expand Hard disk and ensure "Turn off hard disk after" is set to 0. Click Apply and OK.
3

Kiosk Automation

  1. Press Win + R to open Run. Type netplwiz and press Enter.
  2. Uncheck Users must enter a username and password to use this computer. Click Apply. Enter Username: Tarus and Password: Tarus123. Click OK.
    Note: If the checkbox is missing, go to Settings > Accounts > Sign-in options, and turn off "Require Windows Hello sign-in for Microsoft accounts".
  3. Open Remote Desktop Connection. For Computer, enter [JobNum]-Column. Click Show Options.
  4. Go to the Display tab and check Use all my monitors for the remote session. Go back to the General tab, click Save As..., and save it to the Desktop as Connect to Machine Control.
  5. Right-click the new Desktop shortcut and select Properties. In the Target field, move your cursor to the very beginning and paste the following high-priority command before the file path: 
    cmd /c start /high mstsc.exe "C:\Users\Tarus\Desktop\Connect to Machine Control.rdp"
  6. Press Win + R, type shell:startup, and hit Enter. Drag (or Cut/Paste) the modified shortcut from the Desktop into this Startup folder.
  7. From your downloaded Tarus Tech folder, copy the TARUS Desk PC Status Message logo.png file into the Pictures folder and set it as the Desktop Background.
  8. Right-click the Desktop, select View, and uncheck Show desktop icons for a clean kiosk appearance. Restart the PC to verify it logs in and connects automatically.
Support & Diagnostics

Connection Troubleshooting

Resolves common network and video latency issues preventing communication.

Network Stack Corruption

If Remote Desktop fails despite verifying Ethernet cables and pinging the IP successfully, the Windows networking framework may be corrupt.

  1. Launch WinUtil (Win + R > PowerShell > irm christitus.com/win | iex)
  2. Navigate to the Config Tab.
  3. Click Reset Network under Fixes.
  4. Restart PC and re-apply static IPv4 settings.

High Latency / Video Lag

If the Desk PC connects but the interface is severely lagging or dropping frames, check the Column PC graphics settings.

  1. Go to Settings > System > Display > Graphics settings on the Column PC.
  2. Ensure Hardware-accelerated GPU scheduling is OFF.
  3. Ensure TarusNC.exe is set to High Performance.
